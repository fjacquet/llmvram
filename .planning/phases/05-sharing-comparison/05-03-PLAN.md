**2. Modify `src/components/layout/ResultsPanel.tsx`:**

Add a "Save to Compare" button in the results section (State 4: Results). Place it at the top of the results card, in a row with the "VRAM Requirements" heading.

Implementation:
- Import `useComparisonStore` and `ConfigSnapshot` from `@store/comparisonStore`.
- Read `snapshots` and `addSnapshot` from comparison store.
- Create a `handleSaveToComparison` function that:
  1. Extracts config from current store state (selectedModel, selectedGPU, quantization, etc.)
  2. Converts calculation results from Decimal to plain numbers using `.toNumber()`:
     - `result.vram.total.toNumber()` for totalVRAM
     - `result.vram.modelWeights.toNumber()` for weights
     - Same for kvCache, activations, frameworkOverhead
     - `result.performance.tokensPerSecond.toNumber()` for performance
     - `result.performance.timeToFirstToken.toNumber()` for TTFT
     - `result.performance.bottleneck` (already a string)
     - Compute `fits` boolean from the existing `doesNotFit` variable (invert it: `fits: !doesNotFit`)
     - For multi-GPU: `result.multiGPU?.totalPerGPU.toNumber() ?? null`
     - For multi-GPU: `result.multiGPU?.utilizationPercent.toNumber() ?? null`
  3. Calls `addSnapshot({ label: generateLabel(), config, results })` where `generateLabel()` creates a descriptive name like `"Llama 3 70B / H100 / GPTQ"` using model name, GPU name, and quantization.

- Button placement: Next to the "VRAM Requirements" h2 heading, aligned right.
- Button text: "Save to Compare" with a small icon (use `PlusIcon` from @heroicons/react/24/outline).
- Disabled state: When `snapshots.length >= 3`, show "Max 3 saved" with disabled styling. Also disabled when no result exists.
- On successful save: Show `toast.success('Configuration saved to comparison')`.
- Style: `text-sm px-3 py-1.5 rounded-md bg-blue-50 text-blue-700 hover:bg-blue-100 dark:bg-blue-900/30 dark:text-blue-300 dark:hover:bg-blue-900/50 border border-blue-200 dark:border-blue-800`

The button row for the VRAM section header:
```tsx
<div className="flex items-center justify-between">
  <h2 className="text-xl font-semibold text-gray-900 dark:text-white">
    VRAM Requirements
  </h2>
  <button
    onClick={handleSaveToComparison}
    disabled={snapshots.length >= 3}
    className="..."
  >
    {snapshots.length >= 3 ? 'Max 3 saved' : 'Save to Compare'}
  </button>
</div>
```
