**1. Create `src/store/urlSerializer.ts`:**

Define a Zod schema `URLStateSchema` that describes the serializable config state (NOT dark mode, NOT full model/GPU objects):

```typescript
const URLStateSchema = z.object({
  // Model: ID for curated, full params for custom
  modelId: z.string().optional(),
  customModel: z.object({
    name: z.string(),
    num_parameters_billion: z.number(),
    hidden_size: z.number(),
    num_hidden_layers: z.number(),
    num_attention_heads: z.number(),
    num_kv_heads: z.number().optional(),
    intermediate_size: z.number(),
  }).optional(),
  // GPU: ID for curated, full params for custom
  gpuId: z.string().optional(),
  customGPU: z.object({
    name: z.string(),
    vram_gb: z.number(),
    memory_bandwidth_gbps: z.number(),
    fp16_tflops: z.number().optional(),
  }).optional(),
  // Calculation parameters
  q: z.string(), // quantization (short key to save bytes)
  sl: z.number(), // sequenceLength
  bs: z.number(), // batchSize
  kvq: z.string(), // kvQuantization
  ng: z.number(), // numGPUs
  ss: z.string(), // shardingStrategy
  // Offloading (only if enabled)
  oe: z.boolean().optional(), // offloadingEnabled
  ot: z.string().optional(), // offloadTarget
  om: z.string().optional(), // offloadMode
  op: z.number().optional(), // offloadPercentage
  ol: z.number().optional(), // offloadLayers
  ko: z.boolean().optional(), // kvCacheOffload
})
```

Use SHORT keys (q, sl, bs, etc.) to minimize URL size before compression.

Export `isCustomId(id: string): boolean` — returns true if id starts with `'custom-'`.

Export `serializeToURL(storeState)`: Extract relevant fields from store state, use short keys, determine if model/GPU is custom (check `isCustomId`), JSON.stringify, then `compressToEncodedURIComponent` from lz-string. Return the compressed string.

Export `deserializeFromURL(hash: string)`: `decompressFromEncodedURIComponent`, JSON.parse, validate with `URLStateSchema.safeParse()`. Return parsed state or null on any failure. Never throw.

Import lz-string functions: `import { compressToEncodedURIComponent, decompressFromEncodedURIComponent } from 'lz-string'`

**2. Modify `src/store/uiStore.ts`:**

Add two exported helper functions at the module level (before the store):

```typescript
import modelsData from '@data/models.json'
import gpusData from '@data/gpus.json'
import { validateModels, validateGPUs } from '@utils/schemas'

const models = validateModels(modelsData)
const gpus = validateGPUs(gpusData)

export function findModelById(id: string): Model | null {
  return models.find(m => m.id === id) ?? null
}

export function findGPUById(id: string): GPU | null {
  return gpus.find(g => g.id === id) ?? null
}
```

Modify the `partialize` in the existing persist middleware to ONLY persist `isDarkMode` to localStorage. Remove ALL other fields (quantization, sequenceLength, batchSize, etc.) from partialize — those will be handled by URL hash instead. Keep the same localStorage key `'llmvram-ui-preferences'`.

**3. Create `src/hooks/useURLSync.ts`:**

This hook provides bidirectional sync between the Zustand store and URL hash:

On mount (useEffect with empty deps):
1. Read `window.location.hash.slice(1)`
2. If hash exists, call `deserializeFromURL(hash)`
3. If valid, apply state to store:
   - For `modelId`: call `findModelById(modelId)`. If found, `setSelectedModel(model)`. If not found and `customModel` exists, create Model from customModel fields (with id `'custom-restored'`). If neither, show toast via sonner: "Model from shared link not found in database" and leave selectedModel as null.
   - Same pattern for `gpuId`/`customGPU`.
   - Apply all other fields via their respective setters (setQuantization, setSequenceLength, etc.)
   - For offloading fields, only apply if `oe` (offloadingEnabled) is true.

On store change (useEffect with store subscription):
1. Use `useUIStore.subscribe()` to listen for state changes
2. Debounce by 300ms using setTimeout/clearTimeout pattern
3. On debounced change, call `serializeToURL(storeState)` with current state
4. Write to URL via `window.history.replaceState(null, '', '#' + compressed)` (NOT `location.hash =` to avoid hashchange events and history pollution)
5. Before writing, compare new hash with current `location.hash.slice(1)` — skip if identical (avoid infinite loop)
6. If compressed string length > 1800, log `console.warn('URL state exceeds recommended limit')`

Clean up timeout and subscription on unmount.

Import toast from sonner for error notifications.
