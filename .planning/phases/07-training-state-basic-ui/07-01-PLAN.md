    **1. Extend URLStateSchema in `src/store/urlSerializer.ts`:**

    Add these fields to the schema (after offloading fields):
    ```typescript

    m: z.enum(['inference', 'training']).optional(),
    // Training parameters (only present when mode=training)
    tm: z.enum(['full', 'lora', 'qlora']).optional(),       // trainingMethod
    to: z.enum(['adamw', 'sgd-momentum', 'adamw-8bit', 'adafactor']).optional(),  // optimizer
    tp: z.enum(['fp32', 'fp16', 'bf16']).optional(),         // trainingPrecision
    lr: z.number().optional(),                                // loraRank
    la: z.number().optional(),                                // loraAlpha
    tmp: z.number().optional(),                               // targetModulesPercent
    ```

    Short key naming convention matches existing pattern (q, sl, bs, kvq, etc.) — keeps URLs compact before lz-string compression.

    **2. Extend `serializeToURL` function:**

    Add `mode` to the function's state parameter type. Import `FineTuningMethod`, `OptimizerType`, `TrainingPrecision` from `@engines/types`.

    Add training state to the URLState construction. Use mode-conditional serialization to keep inference URLs short:
    ```typescript
    // Training state (only if mode is training)
    ...(state.mode === 'training' ? {
      m: state.mode,
      tm: state.trainingMethod,
      to: state.optimizer,
      tp: state.trainingPrecision,
      lr: state.loraRank,
      la: state.loraAlpha,
      tmp: state.targetModulesPercent,
    } : {}),
    ```

    **3. Extend `useURLSync.ts` hydration:**

    In the mount effect, after restoring offloading parameters, add training state restoration:
    ```typescript
    // Restore training mode and parameters
    if (urlState.m === 'training') {
      store.setMode('training')
      if (urlState.tm) store.setTrainingMethod(urlState.tm as FineTuningMethod)
      if (urlState.to) store.setOptimizer(urlState.to as OptimizerType)
      if (urlState.tp) store.setTrainingPrecision(urlState.tp as TrainingPrecision)
      if (urlState.lr !== undefined) store.setLoraRank(urlState.lr)
      if (urlState.la !== undefined) store.setLoraAlpha(urlState.la)
      if (urlState.tmp !== undefined) store.setTargetModulesPercent(urlState.tmp)
    }
    ```

    Import `FineTuningMethod`, `OptimizerType`, `TrainingPrecision` from `@engines/types`.

    - Deserializing a training URL restores mode + method + optimizer + precision + LoRA params
    - Deserializing an inference URL (no `m` field) defaults to inference mode
    - Backward compatibility: old URLs without `m` field deserialize successfully (no mode field = inference)
