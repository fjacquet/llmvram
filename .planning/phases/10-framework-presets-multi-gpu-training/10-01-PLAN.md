    4. calculateZeROMemoryPerGPU(singleGPU, numGPUs, zeroStage) -> ZeROResult:

       Given a single-GPU TrainingVRAMBreakdown with:
         modelWeights=14, masterWeights=28, gradients=14, optimizerStates=56,
         activations=5, frameworkOverhead=1.5, total=118.5

       ZeRO-1 on 4 GPUs (partition optimizer only):
         perGPU: modelWeights=14, masterWeights=28, gradients=14, optimizerStates=56/4=14,
                 activations=5/4=1.25, frameworkOverhead=1.5, total=72.75
         Expected total: 14 + 28 + 14 + 14 + 1.25 + 1.5 = 72.75

       ZeRO-2 on 4 GPUs (partition optimizer + gradients):
         perGPU: modelWeights=14, masterWeights=28, gradients=14/4=3.5, optimizerStates=56/4=14,
                 activations=5/4=1.25, frameworkOverhead=1.5, total=62.25
         Expected total: 14 + 28 + 3.5 + 14 + 1.25 + 1.5 = 62.25

       ZeRO-3 on 4 GPUs (partition all training state):
         perGPU: modelWeights=14/4=3.5, masterWeights=28/4=7, gradients=14/4=3.5,
                 optimizerStates=56/4=14, activations=5/4=1.25, frameworkOverhead=1.5*1.15=1.725
         Expected total: 3.5 + 7 + 3.5 + 14 + 1.25 + 1.725 = 30.975

       1 GPU passthrough: returns original breakdown total unchanged

       2 GPUs, ZeRO-1: optimizerStates/2, activations/2, rest replicated

    5. calculateCPUOffloadMemory(breakdown, config) -> { gpuMemory, cpuMemory }:

       Given breakdown with optimizerStates=56, modelWeights=14, total=118.5:

       offloadOptimizer=true, offloadParameters=false:
         gpuMemory = 118.5 - 56 = 62.5
         cpuMemory = 56

       offloadOptimizer=true, offloadParameters=true:
         gpuMemory = 118.5 - 56 - 14 = 48.5
         cpuMemory = 56 + 14 = 70

       offloadOptimizer=false, offloadParameters=false:
         gpuMemory = 118.5 (unchanged)
         cpuMemory = 0
  </behavior>
