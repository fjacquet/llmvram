**Row building logic** (same conditional logic as chart):

2. If isLoRA: cast to LoRAVRAMBreakdown, add "Base Model Weights" (baseWeights) and "Adapter Weights" (adapterWeights)

**Table structure** (adapt from MemoryBreakdownTable.tsx):
- `<table>` with `aria-label="Training memory breakdown details"` and same Tailwind classes
- thead with Component / Size (GB) / % of Total columns
- tbody with rows.map: color dot + name, value.toFixed(2), percentage via `value.div(breakdown.total).mul(100).toFixed(1)`
- Bold total row with border-t-2

**Additional section below the table** -- trainable parameters info:
```tsx
<div className="mt-3 text-sm text-gray-600 dark:text-gray-400">
  <span className="font-medium">Trainable:</span>{' '}
  {breakdown.trainableParameters.toFixed(3)}B of{' '}
  {breakdown.totalParameters.toFixed(1)}B parameters
  ({trainablePercent.toFixed(1)}%)
</div>
```

```tsx
{isLoRA && (

  </div>
)}
```
